A C++ application for managing multiple playlists with advanced features including:

-Create, copy, and remove playlists

-Add/remove songs (including removing the last song)

-Compare two playlists for similarities/differences

-Find the longest song across all playlists

-Display common and unique songs between playlists

-Play a song by index

-Track total playlists created

-Perfect for practicing STL containers, playlist algorithms, and menu-driven program design.
